## 1. 一个神经元的数学

```
输入 x → 加权求和 → 激活函数 → 输出 y
       x1*w1
x ─────────┐
       x2*w2 ──→ Σ → 激活 → y
x ─────────┘
       bias b
```

```python
# 单个神经元
def neuron(x, w, b):
    raw = sum(xi * wi for xi, wi in zip(x, w)) + b  # 加权求和 + 偏置
    activated = max(0, raw)  # ReLU 激活函数
    return activated

# forward
x = [1.0, 2.0, 3.0]   # 3个输入
w = [0.1, 0.2, 0.3]   # 3个权重
b = 0.5               # 1个偏置

y = neuron(x, w, b)
```

## 2. 一层神经元（全连接层）

一层 = 多个神经元并行计算。

```
输入 [x1, x2, x3]
      ↓
  神经元1 → y1
  神经元2 → y2   （3个神经元）
  神经元3 → y3
      ↓
输出 [y1, y2, y3]
```

```python
# 全连接层 = 矩阵乘法 + 激活
# 输入: [batch, in_features]
# 输出: [batch, out_features]

class LinearLayer:
    def __init__(self, in_features, out_features):
        self.weight = torch.randn(in_features, out_features) * 0.01
        self.bias = torch.zeros(out_features)

    def forward(self, x):
        return x @ self.weight + self.bias

# PyTorch 的实现（一行）
linear = nn.Linear(in_features=128, out_features=256)
output = linear(input_tensor)  # [batch, 128] → [batch, 256]
```

```python
def relu(x):

F.relu(x)
```

```

输出: [ 0,  0, 0, 1, 2]
```

```python
# GELU = 平滑版的 ReLU，更常用在 Transformer 里

```

### Softmax（用于分类）

```python
# 把一堆数字变成概率分布（和为1）
```

## 4. 什么是层归一化（LayerNorm）？

把一层的输出"标准化"到稳定的数值范围。

```python
# LayerNorm：均值=0，标准差=1，然后学一个缩放和偏移
ln = nn.LayerNorm(128)
output = ln(input_tensor)  # 形状不变：[batch, seq, 128]
```

```js
// 类比 JavaScript 的数据标准化
function normalize(arr) {
  const mean = arr.reduce((a, b) => a + b) / arr.length;
  const std = Math.sqrt(arr.map(v => (v - mean) ** 2).reduce((a, b) => a + b) / arr.length);
  return arr.map(v => (v - mean) / std);
}
```

## 5. 什么是 Dropout？

训练时随机"关闭"一些神经元，防止过拟合。

```python
dropout = nn.Dropout(p=0.1)  # 10% 的神经元随机置零
output = dropout(output)      # 训练时生效，推理时自动关闭
```

```js
// 模拟 Dropout
function dropout(arr, rate) {
  return arr.map(v => Math.random() > rate ? v : 0);
}
// 训练时：[1, 2, 3, 0, 5]（随机丢弃一些）
// 推理时：[1, 2, 3, 4, 5]（全部参与）
```

## 6. 前向传播 vs 反向传播

```
前向传播（Forward）：输入 → 一层层计算 → 输出
        ↓
计算损失（Loss）：预测值 vs 真实值 → 损失值
        ↓
反向传播（Backward）：从输出反向追溯 → 每个参数对损失的"贡献"
        ↓
梯度下降：沿着梯度的反方向，调整参数
```

```python
# 完整流程
optimizer.zero_grad()      # 1. 清零梯度
output = model(input)       # 2. 前向传播
loss = loss_fn(output, y)   # 3. 算损失
loss.backward()             # 4. 反向传播（自动计算所有参数的梯度）
optimizer.step()            # 5. 用梯度更新参数
```

## 7. 什么是梯度？

梯度 = 在当前位置，沿着哪个方向走，函数值增加最快。

```
梯度 ∇f(x) 指向函数增长最快的方向
-梯度 指向函数减小最快的方向（下山方向）
```

```python
# 梯度下降：往梯度的反方向走
new_param = param - learning_rate * gradient

# 类比爬山
# 梯度 = 坡度最陡的方向
# 往梯度的反方向走 = 往下山的方向走
# 学习率 = 每一步迈多远
```

---

## 8. 什么是学习率？

学习率（Learning Rate）= 每一步更新的幅度。

| 学习率 | 效果 |
|--------|------|
| 太大（0.1） | 迈太大，在最优解附近震荡，永远到不了 |
| 太小（1e-6） | 一步走一毫米，收敛极慢 |
| 合适（3e-4） | 稳步下山，收敛到最优解 |

```python
# 学习率衰减（像跑步开始快，后来慢）
scheduler = torch.optim.lr_scheduler.StepLR(optimizer, step_size=100, gamma=0.5)
# 每100步，学习率减半
```

## 9. 什么是反向传播？

反向传播 = 用链式法则，从输出反向计算每个参数的梯度。

```
损失 L
  ↓
∂L/∂输出层参数 = ?
  ↓
∂L/∂中间层参数 = ?
  ↓
∂L/∂输入层参数 = ?
```

```python
# 本项目的反向传播
loss = F.cross_entropy(logits.view(-1, vocab_size), targets.view(-1))
loss.backward()  # 自动计算所有参数的梯度

# 链式法则：dL/dW = dL/dy * dy/dW
# PyTorch 的 autograd 自动做这件事
```

## 10. 什么是优化器？

优化器 = 根据梯度更新参数的算法。

### SGD（随机梯度下降）

```python
# 最简单：param = param - lr * gradient
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)
```

### AdamW（本项目用的）

AdamW = Adam + 权重衰减，是目前最常用的优化器。

```python
optimizer = torch.optim.AdamW(
    model.parameters(),
    lr=3e-4,           # 学习率（本项目默认）
    weight_decay=0.1,  # 权重衰减（正则化，防止过拟合）
    betas=(0.9, 0.95) # 动量参数
)
```

AdamW 的优势：
- 自动调整每个参数的学习率（大梯度用小学习率）
- 收敛快
- 带权重衰减（比 L2 正则化更好）

## 11. 什么是 batch？

batch = 一次喂给模型多少条样本。

```python
batch_size = 32

# batch_size=32 的意思是：
# 一次喂 32 个样本进去，算 32 次前向传播
# 然后把这 32 次的损失加起来，算一次梯度，更新一次参数
```

| batch_size | 优点 | 缺点 |
|------------|------|------|
| 大（512） | 梯度稳定，并行计算快 | 显存占用大 |
| 小（8） | 显存小，更新频繁 | 梯度噪声大，不稳定 |
| 适中（32） | 平衡之选 | — |

## 12. 什么是 epoch？

epoch = 把整个训练集过一遍。

```python
# 假设有 1000 条数据，batch_size=32
# 每个 epoch 有 1000/32 ≈ 31 个 step

for epoch in range(10):     # 训练 10 个 epoch
    for step in range(31): # 每个 epoch 过一遍数据
        x, y = get_batch()
        loss = model(x, y)
        ...
```

本项目**没有用 epoch**，而是用 `max_steps`：

```python
for step in range(max_steps):  # 直接按步数训练，不按 epoch
    x, y = batcher.get_batch("train")
    ...
```
