# 深度学习概览

> 深度学习是机器学习的一个分支，核心是用多层神经网络从数据里自动学习特征。

---

## 1. 深度学习和机器学习的关系

```
人工智能 AI
  ↓
机器学习 Machine Learning
  ↓
深度学习 Deep Learning
  ↓
大模型 Large Language Model / Multimodal Model
```

简单说：

| 概念 | 重点 |
|---|---|
| 人工智能 | 让机器表现出智能行为 |
| 机器学习 | 从数据里学习规律 |
| 深度学习 | 用多层神经网络学习复杂规律 |
| 大模型 | 用大规模数据和参数训练出的通用深度学习模型 |

---

## 2. 为什么深度学习重要？

传统机器学习经常需要人工设计特征。

```
图片分类：
传统方法：人工设计边缘、纹理、颜色特征
深度学习：模型自己从像素里学特征
```

文本也是一样：

```
传统方法：人工设计关键词、词频、规则
深度学习：模型自己学习词义、上下文、语义关系
```

深度学习厉害的地方：

- 能处理非结构化数据：文本、图片、语音、视频
- 能自动学习多层特征
- 数据和算力足够时，效果会持续提升
- 可以迁移到很多任务上

---

## 3. 深度学习的典型任务

### 3.1 视觉任务

| 任务 | 例子 |
|---|---|
| 图像分类 | 判断图片是猫还是狗 |
| 目标检测 | 找出图里的车、人、红绿灯 |
| 图像分割 | 给每个像素分类 |
| 图像生成 | 文生图、图生图 |

常见模型：

```
CNN、ResNet、ViT、Diffusion
```

### 3.2 语音任务

| 任务 | 例子 |
|---|---|
| ASR | 语音转文字 |
| TTS | 文字转语音 |
| 声纹识别 | 判断是谁在说话 |

常见模型：

```
RNN、Transformer、Conformer
```

### 3.3 自然语言处理

| 任务 | 例子 |
|---|---|
| 文本分类 | 情感分析、垃圾识别 |
| 信息抽取 | 从合同里抽字段 |
| 机器翻译 | 中文转英文 |
| 文本生成 | 写文章、写代码 |
| 问答系统 | 根据文档回答问题 |

常见模型：

```
BERT、GPT、T5、LLaMA、Qwen
```

### 3.4 多模态任务

多模态 = 同时处理文字、图片、音频、视频等。

例子：

- 看图回答问题
- 图片生成描述
- 视频理解
- 文档截图理解
- 语音对话助手

## 4. 深度学习训练流程

```
准备数据
  ↓
数据清洗和预处理
  ↓
构建模型
  ↓
选择损失函数
  ↓
选择优化器
  ↓
训练
  ↓
验证和调参
  ↓
保存模型
  ↓
部署推理
```

本项目对应：

| 通用流程 | tinyllm |
|---|---|
| 准备数据 | `data/sample.txt` |
| 预处理 | tokenizer encode |
| 构建模型 | `GPT(GPTConfig)` |
| 损失函数 | cross entropy |
| 优化器 | AdamW |
| 训练 | `tinyllm/train.py` |
| 验证 | `estimate_loss()` |
| 保存模型 | checkpoint |
| 推理 | `tinyllm/generate.py` |

## 5. 深度学习为什么需要 GPU？

神经网络训练主要是大量矩阵乘法。

```
CPU：适合复杂逻辑、少量并行
GPU：适合大量相同计算并行
```

神经网络里的矩阵运算非常适合 GPU：

```python
output = x @ weight + bias
```

这个操作会在训练中反复出现。模型越大、数据越多，GPU 的优势越明显。

本项目默认很小，所以 CPU 也能跑；真实大模型训练通常需要大量 GPU 集群。

| 角色 | 关注点 |
|---|---|
| 算法工程师 | 模型结构、训练、评估 |

## 8. 深度学习常见误区

### 误区 1：模型越大一定越好

不一定。数据质量、任务场景、推理成本都很重要。

### 误区 2：loss 低就代表模型好

不一定。训练集 loss 低可能只是过拟合。还要看验证集、真实任务表现。

### 误区 3：大模型能替代所有算法

不一定。表格数据、小样本预测、强规则任务，传统算法可能更好、更便宜。

### 误区 4：会调 API 就等于懂大模型

调 API 是应用入口，但理解 tokenizer、attention、上下文、采样、评估，会让你做出更可靠的应用。

## 9. 和大模型的关系

大模型是深度学习发展到大规模后的结果。

```
神经网络
  ↓
深度神经网络
  ↓
Transformer
  ↓
大语言模型
  ↓
多模态大模型 / Agent
```

本项目把这条链里的关键一段做小：

```
Transformer → GPT → 语言模型 → 生成
```

你先把这段学透，再去看 Hugging Face、RAG、Agent，会顺很多。
