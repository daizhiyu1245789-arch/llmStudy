# 常用机器学习算法

> 这一篇不是让你马上精通所有算法，而是先建立一张“算法地图”：遇到问题时知道有哪些工具。

---

## 1. 先按任务类型分类

机器学习算法通常先看任务类型：

| 任务 | 目标 | 例子 |
|---|---|---|
| 回归 | 预测连续数值 | 房价、销量、温度 |
| 分类 | 预测类别 | 垃圾邮件、情感分类 |
| 聚类 | 自动分组 | 用户分群、主题发现 |
| 降维 | 压缩特征 | 可视化、去噪 |
| 序列建模 | 处理顺序数据 | 文本、语音、时间序列 |
| 生成 | 生成新内容 | 文本、图片、音乐 |
| 强化学习 | 通过试错学策略 | 游戏、机器人、推荐策略 |

## 2. 回归算法

### 2.1 线性回归

最基础的预测数值算法。

```
y = w1*x1 + w2*x2 + ... + b
```

适合：

- 房价预测
- 销量预测
- 简单趋势拟合

优点：

- 简单
- 快
- 可解释性强

缺点：

- 只能表达线性关系
- 对复杂模式能力弱

### 2.2 多项式回归

在线性回归基础上加入平方、立方等特征。

```
y = w1*x + w2*x² + w3*x³ + b
```

适合有弯曲趋势的数据，但特征设计仍然需要人参与。

## 3. 分类算法

### 3.1 逻辑回归

名字里有“回归”，但常用于二分类。

```
输入特征 → 线性打分 → sigmoid → 概率
```

适合：

- 是否垃圾邮件
- 是否流失用户
- 是否点击广告

### 3.2 KNN

KNN 的思想很直观：

```
看离我最近的 K 个样本
它们大多是什么类别，我就是什么类别
```

优点：

- 易理解
- 不需要复杂训练

缺点：

- 数据大时预测慢
- 对特征尺度敏感

### 3.3 朴素贝叶斯

常用于文本分类。

```
根据词出现的概率，判断文本属于哪个类别
```

适合：

- 垃圾邮件识别
- 新闻分类
- 简单情感分析

---

## 4. 树模型

### 4.1 决策树

像一连串 if/else。

```
年龄 > 30？
  是 → 收入 > 10000？
  否 → 是否学生？
```

优点：

- 可解释
- 能处理非线性
- 对特征尺度不敏感

缺点：

- 容易过拟合

### 4.2 随机森林

很多棵决策树一起投票。

```
树1：正类
树2：负类
树3：正类
最终：正类
```

优点：

- 稳定
- 不容易过拟合
- 常作为传统机器学习强基线

### 4.3 GBDT / XGBoost / LightGBM

一类非常强的表格数据算法。

核心思想：

```
一棵树学一点
下一棵树重点修正前一棵树的错误
多棵树叠加，效果越来越好
```

适合：

- 风控
- 推荐排序特征
- 广告点击率
- 表格比赛

在结构化表格数据上，树模型经常比神经网络更省事、更稳。

## 5. 聚类算法

### 5.1 K-Means

把样本分成 K 群。

```
随机选 K 个中心
每个点归到最近中心
重新计算中心
反复迭代
```

适合：

- 用户分群
- 商品分组
- 简单主题聚类

### 5.2 DBSCAN

根据密度聚类，不需要指定类别数量。

适合：

- 异常点检测
- 不规则形状聚类

---

## 6. 降维算法

### 6.1 PCA

把高维数据压到低维，同时尽量保留主要信息。

适合：

- 特征压缩
- 可视化
- 去噪

### 6.2 t-SNE / UMAP

常用于把高维向量画成二维图。

在大模型应用里，你可能会用它们观察 embedding 是否按语义聚在一起。

## 7. 深度学习算法

| 算法/结构 | 适合任务 |
|---|---|
| MLP | 表格、小型分类/回归 |
| CNN | 图像、局部模式识别 |
| RNN / LSTM / GRU | 早期文本、语音、时间序列 |
| Transformer | 文本、多模态、大模型 |
| AutoEncoder | 降维、去噪、异常检测 |
| GAN | 图像生成、风格迁移 |
| Diffusion | 高质量图像/视频生成 |

对于大模型学习，重点优先级：

```
Transformer > MLP > CNN/RNN > AutoEncoder/GAN/Diffusion
```

## 8. 算法怎么选？

### 数据是表格

```
优先：LightGBM / XGBoost / RandomForest
简单基线：Linear / Logistic Regression
```

### 数据是图片

```
优先：CNN / ViT
生成图片：Diffusion
```

### 数据是文本

```
传统分类：朴素贝叶斯 / 逻辑回归 / BERT
文本生成：GPT / Transformer
文档问答：Embedding + 向量检索 + LLM
```

### 数据是时间序列

```
简单预测：线性模型 / 树模型
复杂模式：RNN / Transformer
```

### 需要一个智能体试错

```
强化学习：Q-Learning / DQN / PPO
```

```

    → 语言模型
      → GPT 风格 Transformer
```

- 做表格预测，不一定要用大模型

- 做文档问答，常常需要 embedding 和检索算法
- 做 Agent，可能会接触强化学习思想，但工程上更多是工具调用和规划
