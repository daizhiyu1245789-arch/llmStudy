# 参考答案目录

> 这里是 `07_练习/` 的参考答案。建议先自己做，再来看答案。答案不是唯一写法，重点是检查思路、边界和验收点。

---

## 使用方式

推荐：

```text
先做练习
  ↓
卡住 15-30 分钟
  ↓
看参考答案
  ↓
合上答案自己再写一遍
```

不要直接复制答案。你真正需要的是：

- 为什么这么写
- 边界在哪里
- 怎么验收
- 后面怎么迁移到项目里
