## 03_Attention 机制

### Attention 公式解释

```text
Attention(Q, K, V) = softmax(QK^T / √d) V
```

逐步解释：

```text
QK^T：计算每个 token 和其他 token 的匹配分数
/ √d：缩放，避免分数太大
softmax：把分数变成注意力概率
乘 V：按注意力概率汇总信息
```

### Q/K/V 直觉

```text
Q：我现在想找什么
K：我能被别人怎么找到
V：我真正携带的信息
```

## 04_Tokenizer 分词器

### 字符级 token 示例

文本：

```text
今天天气很好
```

字符列表：

```text
["今", "天", "天", "气", "很", "好"]
```

词表：

```text
今 → 1
天 → 2
气 → 3
很 → 4
好 → 5
```

编码：

```text
[1, 2, 2, 3, 4, 5]
```

解码：

```text
今天天气很好
```

### 完整链路

```text

  ↓ tokenizer.py

  ↓ model.py / wte
Embedding
  ↓ model.py / Block
Transformer
  ↓ model.py / lm_head
logits
  ↓ generate.py / sampling
next token
  ↓ tokenizer.py
文本
```
