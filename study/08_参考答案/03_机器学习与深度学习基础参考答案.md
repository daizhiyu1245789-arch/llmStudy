## 00_大模型数学直觉

### 关键解释

```text
标量：一个数字，比如 loss=2.1
向量：一串数字，比如 embedding=[0.1, -0.2, ...]
矩阵：二维数字表，比如一批 token 的向量
张量：更高维数组，比如 [batch, seq, hidden]
```

### logits 和 probability

```text
logits：模型输出的原始分数，不一定加起来等于 1。
probability：经过 softmax 后的概率，加起来等于 1。
```

### cosine similarity

```ts
function cosineSimilarity(a: number[], b: number[]): number {
  let dot = 0;
  let normA = 0;
  let normB = 0;

  for (let i = 0; i < a.length; i++) {
    dot += a[i] * b[i];
    normA += a[i] * a[i];
    normB += b[i] * b[i];
  }

  return dot / (Math.sqrt(normA) * Math.sqrt(normB));
}

console.log(cosineSimilarity([1, 0], [1, 0])); // 1
console.log(cosineSimilarity([1, 0], [0, 1])); // 0
```

## 01_机器学习是什么

### 训练流程伪代码

```text
for batch in data:
  pred = model(x)              # 前向传播
  loss = loss_fn(pred, y)      # 计算损失
  loss.backward()              # 反向传播，计算梯度
  optimizer.step()             # 更新参数
  optimizer.zero_grad()        # 清空梯度
```

### 关键解释

```text
模型 = 参数化函数
训练 = 找到更好的参数
预测 = 用已有参数输出结果
loss = 错误程度
```

## 02_神经网络基础

### 两层 MLP

```python
import torch
import torch.nn as nn
import torch.nn.functional as F


class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(128, 256)
        self.fc2 = nn.Linear(256, 10)

    def forward(self, x):
        x = self.fc1(x)
        x = F.gelu(x)
        x = self.fc2(x)
        return x


model = MLP()
x = torch.randn(32, 128)
y = model(x)
print(y.shape)  # [32, 10]
```

## 06_神经网络进阶

### 问题排查表

| 现象 | 可能原因 | 解决办法 |
|---|---|---|
| loss 不下降 | 学习率不合适、数据太少、模型太小 | 调学习率、检查数据、训练更久 |
| train loss 低但 val loss 高 | 过拟合 | 加数据、dropout、weight decay |
| loss NaN | 梯度爆炸、学习率太大 | 降学习率、开 grad_clip |
| 生成很乱 | 训练不足、temperature 太高 | 多训练、降低 temperature |

## 09_强化学习入门

### 走迷宫例子

```text
Agent：走迷宫的小人
Environment：迷宫
State：当前位置、墙、终点
Action：上、下、左、右
Reward：到终点 +10，撞墙 -1，每走一步 -0.1
```

### 关键结论

```text
强化学习里的 Agent 是通过奖励学习策略。
工程里的 LLM Agent 通常是大模型 + 工具调用 + 执行循环。
两者有关，但不是一回事。
```
