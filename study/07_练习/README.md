# 练习目录

> 这里是配套练习。建议你每读完一个章节，就来这里完成对应练习。不要急着“看完全部”，真正的掌握来自动手。

---

## 怎么使用？

推荐节奏：

```text
读一个章节
  ↓
做对应练习
  ↓
对照验收标准
  ↓
记录卡点
  ↓
再进入下一章
```

每个练习分三层：

- 基础练习：必须完成，用来确认你看懂了
- 实战练习：动手写代码或跑命令
- 加分挑战：做完会更接近真实项目

## 建议打卡方式

每次练习完成后，在自己的笔记里写：

```text
章节：
我完成了：
我卡住了：
我用自己的话解释：
下一步：
```

如果你能把一个概念讲给别人听，并且能写出最小代码，那就是真会了。
