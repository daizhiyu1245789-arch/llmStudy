# 学习目录

> Node / 前端工程师学大模型，跟着这个目录一步步走，从零到能跑起来、改起来、接入真实模型、部署起来。

如果你是 Node 开发者，建议先读：

- `00_Node开发者学习总路线.md`
- `00_大模型术语表.md`
- `00_学习打卡和项目作业.md`

---

## 学习路线总览

```
阶段一：Python + PyTorch 基础（1-2周）
    → Node 工程基础、Python 代码看得懂，脚本跑得起来
    ↓
阶段二：深度学习核心概念（1-2周）
    → 神经网络、常用算法、深度学习、强化学习、Transformer 搞懂
    ↓
阶段三：项目实战（1周）
    → 跑 tinyllm、接 Hugging Face、做 Prompt/RAG/工具调用
    ↓
阶段四：扩展学习（持续）
    → 读论文、用 HuggingFace、部署上线
```

## 建议学习顺序

### 第一步：Node 应用基础（2-4天）

```
1. 读 01_Node应用基础/README.md
   → 知道 Node 大模型应用需要哪些工程基础

2. 读 01_Node应用基础/01_TypeScript工程基础.md
   → 类型、接口、联合类型、Zod 参数校验

3. 读 01_Node应用基础/02_HTTP接口和SSE流式响应.md
   → 普通 API 和流式输出，理解 ChatGPT 式打字效果

4. 读 01_Node应用基础/03_环境变量和密钥安全.md
   → 学会保护 HF_TOKEN、API Key、用户隐私

5. 读 01_Node应用基础/04_异步编程和错误处理.md
   → Promise、try/catch、超时、重试、requestId

6. 读 01_Node应用基础/05_文件处理和文本预处理.md
   → 为 RAG 做文件读取、文本清洗、切块

7. 读 01_Node应用基础/06_Node大模型应用项目结构.md
   → 学会 route/service/provider/prompt/store/tool 分层
```

### 第二步：Python 基础（1-3天）

```
8. 读 01_Python基础/01_Python_vs_JS语法对比.md
   → 熟悉 Python 基本语法

9. 读 01_Python基础/02_Python独有概念.md
   → 学会缩进、装饰器、with 语句

10. 读 01_Python基础/03_PyTorch快速入门.md
   → 理解张量、nn.Module、GPU 加速

11. 读 01_Python基础/04_动手跑第一个模型.md
   → 把 tinyllm 当作第一个"跑起来的项目"
```

### 第三步：深度学习核心（7-14天）

```
12. 读 02_深度学习核心/00_大模型数学直觉.md
   → 建立向量、矩阵、概率、softmax、loss、相似度的直觉

13. 读 02_深度学习核心/01_机器学习是什么.md
   → 搞懂 ML 的核心思想（模型=函数，训练=调参）

14. 读 02_深度学习核心/02_神经网络基础.md
   → 神经元、反向传播、梯度下降、AdamW

15. 读 02_深度学习核心/06_神经网络进阶.md
   → 补充 MLP、CNN、RNN、Transformer、过拟合、超参数等实战概念

16. 读 02_深度学习核心/07_常用机器学习算法.md
   → 建立回归、分类、聚类、树模型、深度学习算法地图

17. 读 02_深度学习核心/08_深度学习概览.md
   → 理解深度学习的任务、训练流程、框架和工程角色

18. 读 02_深度学习核心/09_强化学习入门.md
   → 理解 Agent、状态、动作、奖励、RLHF 和大模型对齐

19. 读 02_深度学习核心/03_Attention机制.md
   → Transformer 的核心，读懂 model.py 的一半

20. 读 02_深度学习核心/04_Tokenizer分词器.md
   → 文本怎么变成数字

21. 读 02_深度学习核心/05_大模型入门.md
   → 把 Tokenizer、Attention、训练、生成串成一张完整地图
```

### 第四步：项目实战（3-7天）

```
22. 读 03_项目实战/01_把tinyllm跑起来.md
   → 按步骤把训练跑起来，第一次看到模型输出

23. 读 03_项目实战/02_用自己的数据训练.md
    → 准备自己的语料，看模型学到了什么

24. 读 03_项目实战/03_部署成API服务.md
    → 把模型封装成 HTTP 接口，前端能调

25. 读 03_项目实战/04_Node集成HuggingFace.md
    → 用 Node + Express 接 Hugging Face，支持普通生成和流式输出

26. 读 03_项目实战/05_Prompt模板和结构化输出.md
    → 把 Prompt 当接口协议设计，学会 JSON 输出和 Zod 校验

27. 读 03_项目实战/06_Embedding和向量检索.md
    → 理解文本向量、相似度搜索、文档切块和向量库

28. 读 03_项目实战/07_RAG文档问答.md
    → 做文档问答：检索相关资料，再让模型基于资料回答

29. 读 03_项目实战/08_ToolCalling工具调用.md
    → 让模型选择调用后端函数，理解工程 Agent 的核心

30. 读 03_项目实战/09_日志评估和部署.md
    → 补齐日志、评估、限流、缓存、成本控制和部署
```

### 第五步：扩展（持续）

```
31. 读 04_扩展学习/01_必读论文.md
    → 读 GPT-2 / Attention is All You Need 原文

32. 读 04_扩展学习/02_HuggingFace入门.md
    → 学会用全球最大的模型库

33. 读 04_扩展学习/03_Node开发者大模型学习路线.md
    → 从 Node 开发者视角补齐流式输出、RAG、工具调用、评估和部署
```
