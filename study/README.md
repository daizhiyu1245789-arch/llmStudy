```
阶段一：Python + PyTorch 基础（1-2周）

    ↓
阶段二：深度学习核心概念（1-2周）
    → Attention、Transformer 彻底搞懂
    ↓
阶段三：项目实战（1周）
    → 把 tinyllm 项目跑起来、改自己的数据
    ↓
阶段四：扩展学习（持续）
    → 读论文、用 HuggingFace、部署上线
```

## 建议学习顺序

### 第一步：Python 基础（1-3天）

```
1. 读 01_Python基础/01_Python_vs_JS语法对比.md
   → 熟悉 Python 基本语法

2. 读 01_Python基础/02_Python独有概念.md
   → 学会缩进、装饰器、with 语句

3. 读 01_Python基础/03_PyTorch快速入门.md
   → 理解张量、nn.Module、GPU 加速

4. 读 01_Python基础/04_动手跑第一个模型.md
   → 把 tinyllm 当作第一个"跑起来的项目"
```

### 第二步：深度学习核心（7-14天）

```
5. 读 02_深度学习核心/01_机器学习是什么.md
   → 搞懂 ML 的核心思想（模型=函数，训练=调参）

6. 读 02_深度学习核心/02_神经网络基础.md
   → 神经元、反向传播、梯度下降、AdamW

7. 读 02_深度学习核心/03_Attention机制.md
   → Transformer 的核心，读懂 model.py 的一半

8. 读 02_深度学习核心/04_Tokenizer分词器.md
   → 文本怎么变成数字
```

### 第三步：项目实战（3-7天）

```
9. 读 03_项目实战/01_把tinyllm跑起来.md
   → 按步骤把训练跑起来，第一次看到模型输出

10. 读 03_项目实战/02_用自己的数据训练.md
    → 准备自己的语料，看模型学到了什么

11. 读 03_项目实战/03_部署成API服务.md
    → 把模型封装成 HTTP 接口，前端能调
```

### 第四步：扩展（持续）

```
12. 读 04_扩展学习/01_必读论文.md
    → 读 GPT-2 / Attention is All You Need 原文

13. 读 04_扩展学习/02_HuggingFace入门.md
    → 学会用全球最大的模型库
```
