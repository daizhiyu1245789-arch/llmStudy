# 大模型学习目录

> 这是一套给 Node 开发者准备的大模型入门路线。目标不是一口气变成算法专家，而是从“看得懂、跑得动、改得了、接得上、做得出项目”开始，一层一层往上学。

如果你是小白，先读这 4 个文件：

1. `00_学习导航/00_由浅入深学习地图.md`
2. `00_学习导航/00_Node开发者学习总路线.md`
3. `00_学习导航/00_大模型术语表.md`
4. `00_学习导航/00_学习打卡和项目作业.md`

---

## 总路线

```text
L0 学习导航
   先知道怎么学、怎么做练习、遇到术语去哪查

L1 Node 应用工程基础
   补齐 TypeScript、HTTP、SSE、密钥、异步、文件处理、项目分层

L2 Python 和 PyTorch 基础
   能看懂 tinyllm 代码，知道张量、模型、训练脚本在做什么

L3 机器学习与深度学习基础
   建立数学直觉，理解神经网络、常用算法、深度学习、强化学习

L4 大模型核心原理
   理解 Tokenizer、Attention、Transformer、GPT 生成流程

L5 大模型应用实战
   跑 tinyllm，接 Hugging Face，做 Prompt、Embedding、RAG、Tool Calling、部署

L6 扩展学习
   读论文、学 Hugging Face 生态、继续补齐 Node 大模型工程能力

L7 练习
   每个阶段都有题目，逼自己动手写、画图、解释、改代码

L8 参考答案
   做完后对答案，检查思路、边界、工程写法
```

## 怎么用练习和答案

建议每章按这个节奏来：

1. 先读正文。
2. 合上文档，用自己的话复述一遍。
3. 去 `07_练习/` 做对应练习。
4. 做完再看 `08_参考答案/`。
5. 把答案里你没想到的点，补到自己的笔记里。

不要一开始就看答案。大模型学习最怕“看懂了但不会写”，练习就是专门用来解决这个问题的。
